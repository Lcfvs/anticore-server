# @lcf.vs/dom-engine

**[ALPHA]** An [anticore](https://github.com/Lcfvs/anticore) server example, based on [@lcf.vs/dom-engine](https://github.com/Lcfvs/dom-engine)

## Install

```shell script
mkdir anticore-server
cd anticore-server
git clone https://github.com/Lcfvs/anticore-server.git .
npm i
npm run build
```

Then go to http://localhost:1443/fallback

## Disable the generator

Modify the `.env` file containing this variable

```shell script
NODE_ENV=production

```
